# Yireo_ResetMysqlTriggers

**This Magento 2 module is an add-on package to the LokiCheckout.**

### Installation
Install this package via composer:
```bash
composer require yireo/magento2-reset-mysql-triggers
```

Next, enable this module:
```bash
bin/magento module:enable Yireo_ResetMysqlTriggers
```
